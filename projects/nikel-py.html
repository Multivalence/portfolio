Project Overview

A Python API Wrapper that interacts with the Nikel API to gather data about the University of Toronto such as Courses, Programs, Textbooks, and more. Uses concurrent libraries to allow for asynchronous functionality.

Includes Documentation and TravisCI to test code on deployment and check for version compatibility.
